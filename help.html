使い方ガイド

基本操作

カードの追加

キャンバス上の何もないところをダブルクリックするか、画面下部のツールバーから「カード追加」ボタンを押します。

カードの接続

カードの左右にある丸いポイントをドラッグして、別のカードにドロップすると線でつながります。

移動・ズーム

キャンバスをドラッグして移動、マウスホイールで拡大縮小ができます。

範囲選択

Shiftキーを押しながらドラッグすると、複数のカードをまとめて選択できます。

ショートカットキー

機能 Windows / Linux macOS
保存 Ctrl + S Cmd + S
元に戻す (Undo) Ctrl + Z Cmd + Z
やり直し (Redo) Ctrl + Y / Ctrl + Shift + Z Cmd + Y / Cmd + Shift + Z
すべて選択 Ctrl + A Cmd + A
コピー Ctrl + C Cmd + C
貼り付け Ctrl + V Cmd + V
削除 Delete / Backspace Delete / Backspace
メニューを閉じる Escape Escape

プレゼンテーションモード

作成したカードのつながりをスライドショーとして再生できます。開始したいカードを選択してから「プレゼン」ボタンを押してください。

  • → 次のスライドへ進む
  • ← 前のスライドに戻る
  • Esc プレゼンテーションを終了
